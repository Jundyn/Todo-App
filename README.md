# React To-Do List Application

A simple **To-Do List** application built with React that allows users to **add, edit, delete, and mark tasks as completed**. Tasks are persisted using **localStorage** so that they remain between sessions.

---

## Features

- Add new tasks with a **name** and **description**
- Edit existing tasks
- Delete tasks with confirmation
- Mark tasks as completed (visually distinguished)
- Persist tasks using **localStorage**
- Form validation to ensure task fields are not empty
- Clean, reusable, and well-structured React components
- Basic CSS styling for a user-friendly interface

---

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/todo-app.git
```
2. Navigating to the Project folder

cd todo-app

3. Install dependencies
```bash
npm install
# or
yarn install
```
### Running the app
```bash
npm start
# or
yarn start
```
Open your browser and go to http://localhost:3000 a to see the application.

Usage
1. Add a Task: Fill in the task name and description, then click Add Task.
2. Edit a Task: Click the Edit button on a task, update the fields, and click Update Task.
3. Delete a Task: Click the Delete button. You will be prompted for confirmation.
4. Mark as Completed: Click on the task itself to toggle completion status.
5. Persistence: All tasks are saved in your browser's localStorage and loaded automatically on page refresh.

Optional Enhancements
. Filter tasks by All / Active / Completed
. Sort tasks by priority or due date
. Add categories or due dates for tasks
. Improve styling with CSS frameworks or animations
